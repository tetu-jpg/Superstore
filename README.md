# Sales data analysis project
The project works on sales and profit data of superstore or retail company.

main tasks performed are:-
  exploring dataset
  Data cleaning
  feature engineering
  Groupby and aggregation
  merging and spliting dataset
  pivot table analysis
  Data visiualization
  
Insights:
  the most profitable category
  the least profitable region
  sales and profit trends over time
  relation between sales and profit

tools:
 jupyter or colab notbook
 python
 pandas
 matplotlib
